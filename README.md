# Installation
```
npm i discord-rick-roll --save
```

<br>

# Setup
```js
// Top of the command file
const Rick = require('discord-rick-roll');
const roll = new Rick();
```

<br>

## Create
```js
roll.create({ msg: message })
```

<br>

## Eval
```js
roll.eval({ msg: message, input: yourInput })
```

<br>
